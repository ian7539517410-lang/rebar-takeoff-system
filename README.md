# 新世代智慧鋼筋撿料系統

這是一個可直接開啟使用的瀏覽器版鋼筋撿料原型，聚焦於核心計算透明化、施工圖說視覺化、局部 3D 編輯與料單同步。

## 開啟方式

直接用瀏覽器開啟：

`E:\鋼筋檢料專案\版本一\index.html`

不需要安裝套件或啟動伺服器。

## 已實作功能

- 梁、柱、牆板三類構件撿料。
- 梁上層筋支援左、中、右三區段配置，並自動拆解貫通筋、左端加伸筋、右端加伸筋料單。
- 連續梁台模組支援柱－梁－柱串接、多跨延伸、快速新增柱梁、柱斷面防呆、梁段手動配筋、多層主筋、截斷距、箍筋分區與腰筋料單。
- f'c、fy、Ld 基準係數、頂層筋係數、乙級搭接係數、長度歸整、續接制度可即時修改。
- 專案搭接表可覆寫特定號數的甲級或乙級搭接長度。
- 自動計算 Ld、Ls、總長度與總重量。
- 2D 施工圖自動標示可搭接區、避開區、警示區、主筋與箍筋料號。
- 局部 3D 視圖可拖曳黑色續接器，2D 圖說與料單會同步更新。
- 自動產生圖料一對一料單，可匯出 CSV。
- 可列印或用瀏覽器另存 PDF。
- 修改紀錄區分系統自動生成、參數修改與 3D 手動微調。
- 本機瀏覽器儲存與載入專案。

## 工程使用提醒

目前計算引擎採透明參數化公式：

`Ld = Ld基準係數 * db * (fy / 420) * sqrt(28 / f'c) * 位置係數`

`Ls = Ld * 搭接等級係數`

正式工程使用前，請由結構技師依專案適用的台灣 RC 規範、標準圖與審查要求調整公式係數或專案搭接表。
